Завдання полягало в написанні програми, яка кожні 5 секунд виводить поточний час, поки не 
завершиться інша задача, яка збільшує лічильник на 1 кожні 2 секунди до значення 10. Це 
було реалізовано за допомогою асинхронного виконання задач, де одна задача перевіряє умову 
завершення іншої і виводить час, поки лічильник не досягне 10. Програма використовує 
бібліотеку Spring для асинхронного виконання методів та коректної обробки умов зупинки.
